# garmin-3d-topo-builder

Automated pipeline to generate Garmin 3D topo maps from OpenStreetMap data and NASA SRTM 30m elevation data, using mkgmap and splitter. Profile-driven architecture supports building maps for any region.

## Quick Start

```bash
./update_map.sh profiles/Asia/China/Yunnan.conf
```

## Structure

```
garmin-topo-forge/
├── profiles/               # Region profiles (Geofabrik hierarchy)
│   ├── Africa/
│   ├── Asia/China/Yunnan.conf
│   ├── Europe/Germany/Bayern.conf
│   ├── North America/United States/
│   ├── Russia/
│   └── ...                 # ~475 pre-generated profiles
├── generate_profiles.py    # Regenerate all profiles from Geofabrik tree
├── update_map.sh           # Main pipeline script
├── download_1inch_dem.py   # SRTM DEM tile downloader (uses earthaccess)
├── requirements.txt        # Python deps (earthaccess, pyhgtmap)
├── .env                    # Earthdata credentials
├── bin/
│   ├── mkgmap/             # Auto-downloaded mkgmap distribution
│   └── splitter/           # Auto-downloaded splitter distribution
├── data/
│   └── {region}/           # Per-region data
│       ├── dem_1inch/      # SRTM 1" .hgt tiles
│       ├── contours/       # Generated contour PBFs
│       └── osm/            # OSM PBF extract
├── work/
│   └── {region}/           # Per-region build workspace
└── output/
    └── {region}/           # Per-region final output
```

## Prerequisites

- Java 11+ (OpenJDK)
- Python 3
- `wget`, `curl`, `unzip`
- `osmium-tool` (for bounding box extraction from OSM PBF files)
- NASA Earthdata account (register at https://urs.earthdata.nasa.gov/users/new)

### macOS (Homebrew)

```bash
brew install openjdk osmium-tool wget
```

### Ubuntu/Debian

```bash
sudo apt install default-jdk osmium-tool wget curl unzip python3-venv
```

### Earthdata credentials

Create a `.env` file in the project root:

```
EARTHDATA_USERNAME=your_username
EARTHDATA_PASSWORD=your_password
```

Python dependencies (`earthaccess`, `pyhgtmap`) are installed automatically into a virtual environment on first run.

## Usage

### Build a map from a profile

Profiles for all ~475 Geofabrik regions are pre-generated under `profiles/`, organized by continent:

```bash
./update_map.sh profiles/Asia/China/Yunnan.conf
./update_map.sh profiles/Europe/Germany/Bayern.conf
./update_map.sh "profiles/North America/United States/Colorado.conf"
```

### Override contour settings

Use environment variables to override profile contour defaults without editing the file:

```bash
CONTOUR_STEP_OVERRIDE=20 CONTOUR_LINE_CAT_OVERRIDE="200,100" ./update_map.sh profiles/Asia/Nepal.conf
```

### Regenerate profiles

If Geofabrik adds new regions, update `generate_profiles.py` and re-run:

```bash
python3 generate_profiles.py
```

`SERIES_NAME` and `FAMILY_NAME` are auto-derived from `REGION_NAME` but can be overridden in the profile.

The bounding box for DEM download is automatically extracted from the OSM PBF file header using `osmium fileinfo`.

### Selective rebuilds

| Task | Command |
|------|---------|
| Full build | `./update_map.sh profiles/Asia/China/Yunnan.conf` |
| Force DEM re-download | `rm data/Yunnan/dem_1inch/*.hgt && ./update_map.sh profiles/Asia/China/Yunnan.conf` |
| Force contour regeneration | `rm -rf data/Yunnan/contours && ./update_map.sh profiles/Asia/China/Yunnan.conf` |
| Force tool re-download | `rm -rf bin/mkgmap bin/splitter && ./update_map.sh profiles/Asia/China/Yunnan.conf` |

## Pipeline Phases

1. **Tooling** -- checks for `java`, auto-downloads latest mkgmap + splitter
2. **Data** -- downloads OSM extract (always refreshed), sets up Python venv, extracts bounding box from PBF header via `osmium fileinfo`, downloads SRTM DEM tiles for the detected bbox
3. **Contours** -- runs `pyhgtmap` to generate contour line PBFs (cached per region)
4. **Build** -- runs splitter then mkgmap with 30m DEM, contour lines, and routing
5. **Output** -- copies `gmapsupp.img` and `.gmapi` to region output directory

## Output

```
output/{region}/gmapsupp.img   -> Copy to Garmin device SD card under /Garmin/
output/{region}/*.gmapi        -> Open with Garmin BaseCamp on macOS
```
